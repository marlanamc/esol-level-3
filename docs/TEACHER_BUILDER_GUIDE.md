# Teacher Activity Builder - Quick Start Guide

## What is this?

The Teacher Activity Builder is a simple form that helps you create prompts for AI assistants (like ChatGPT, Claude, or Cursor) to generate beautiful educational activities. **No coding knowledge required!**

## How it works

1. **Fill out the form** - Tell us what kind of activity you want
2. **Get a custom prompt** - We generate a detailed prompt for you
3. **Use with AI** - Copy the prompt and paste it into your favorite AI assistant
4. **Get your activity** - The AI creates a complete HTML file ready to use!

## Step-by-Step Instructions

### Step 1: Open the Builder
- Open `teacher-builder.html` in your web browser
- You'll see a simple form with 6 steps

### Step 2: Choose Activity Type
Click on the type of activity you want:
- **Worksheet** - Practice exercises, fill-in-the-blank
- **Quiz** - Multiple choice, answer checkers
- **Game** - Trivia, word scrambles, interactive games
- **Study Guide** - Reference materials, explanations
- **Slides** - Presentation slides, answer keys
- **Interactive** - Clickable exercises, drag-and-drop

### Step 3: Fill in the Details
- **Subject & Topic**: What are you teaching? (e.g., "ESOL - Present Perfect Tense")
- **Level**: Who is this for? (e.g., "Level 3", "Grade 5", "Beginner")
- **Learning Goals**: What should students learn? Be specific!
- **Activity Details**: Describe what students will do
- **Features**: Check any special features you want (examples, hints, answer keys, etc.)

### Step 4: Generate Your Prompt
- Click "‚ú® Generate My Prompt"
- You'll see a detailed prompt appear below

### Step 5: Use with AI
1. Click "üìã Copy Prompt" to copy the text
2. Open your AI assistant:
   - **ChatGPT**: https://chat.openai.com
   - **Claude**: https://claude.ai
   - **Cursor**: Use the AI chat feature
   - Or any other AI assistant
3. Paste the prompt and send it
4. Wait for the AI to generate your activity (usually takes 30-60 seconds)

### Step 6: Save and Use
- The AI will give you complete HTML code
- Copy all the HTML code
- Save it as a `.html` file (e.g., `my-activity.html`)
- Open it in any web browser to use with your students!

## Common Questions

### "What if I don't like what the AI creates?"
Just ask it to modify! You can say:
- "Make it more colorful"
- "Add 10 more questions"
- "Make the font bigger"
- "Change the background color"

### "Can I use this for any subject?"
Yes! Math, Science, History, Language Arts - anything works. Just fill in the subject and topic fields.

### "Do I need an account?"
- For ChatGPT: Yes, you need a free account
- For Claude: Yes, you need a free account
- For Cursor: You need the app installed

### "Is it free?"
The builder itself is free! AI assistants may have free tiers with usage limits, but that's usually enough for creating activities.

### "What if I want something specific?"
Include it in the "Additional Information" section! The more details you provide, the better the result.

## Troubleshooting

### "The prompt is too long"
- That's okay! AI assistants can handle long prompts
- Just copy and paste the whole thing

### "The AI didn't create what I wanted"
- Try being more specific in your description
- Add more details in "Additional Information"
- Ask the AI to modify it: "Can you make it more like [describe what you want]?"

### "I don't understand HTML"
- You don't need to! Just:
  1. Copy the HTML code the AI gives you
  2. Save it as a `.html` file
  3. Open it in a browser
- That's it!

### "Can I edit the activity after?"
- Yes! You can ask the AI to modify it
- Or open the HTML file in a text editor (like Notepad) and make simple changes
- For complex changes, ask the AI to help

## Example Prompts

Here are some examples of what good prompts look like:

### Example 1: Grammar Worksheet
```
Create a beautiful, interactive HTML educational activity for my students.

ACTIVITY TYPE: Worksheet
SUBJECT: ESOL
TOPIC: Present Perfect Tense
LEVEL/GRADE: Level 3

TITLE: Present Perfect Practice Worksheet

LEARNING OBJECTIVES:
Students will practice forming Present Perfect sentences using 'have/has + past participle'. They will learn to use time words like 'already', 'yet', 'just', and 'never'.

ACTIVITY DESCRIPTION:
Students will complete 20 sentences using the Present Perfect tense. Include examples at the top showing the structure. Add a word scramble section with 10 sentences to rearrange.

SPECIAL FEATURES TO INCLUDE:
- Example sentences
- Hints/clues
- Answer key
- Color coding
- Print-friendly
```

### Example 2: Math Quiz Game
```
Create a beautiful, interactive HTML educational activity for my students.

Students will review adding, subtracting, and comparing fractions. They will practice converting between fractions and decimals.

Create a trivia-style game with 15 multiple choice questions about fractions. Include a timer and score counter. Make it visually engaging with colors and animations.

SPECIAL FEATURES TO INCLUDE:
- Interactive elements
```
