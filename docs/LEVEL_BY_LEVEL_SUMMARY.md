# Level-by-Level Summary: Sentence Builder Cards

## Quick Reference: All Available Cards

**Pronouns (6):** I, You, He, She, They, We  
**To Be (10):** am, is, are, was, were, am not, isn't, aren't, wasn't, weren't  
**Helper Verbs (8):** do, does, did, will, don't, doesn't, didn't, won't  
**V1/V1-3rd (17):** eat/eats, work/works, watch/watches, go/goes, study/studies, play/plays, cook/cooks, read/reads, sleep/sleeps, clean/cleans, drive/drives, talk/talks, have/has, do/does, live/lives, need/needs, want/wants  
**V1-ing (17):** eating, working, watching, going, studying, playing, cooking, reading, sleeping, cleaning, driving, talking, having, doing, living, needing, wanting  
**V2 (17):** ate, worked, watched, went, studied, played, cooked, read, slept, cleaned, drove, talked, had, did, lived, needed, wanted  
**Objects (8):** pizza, coffee, English, book, homework, TV, bus, music  
**Time (7):** now, every day, usually, yesterday, last night, tomorrow, next weekend  
**Places (4):** home, school, work, park  
**Prepositions (6):** at, in, on, to, from, with  
**Determiners (5):** a, an, the, my, your  
**Connectors (7):** when, while, before, as, after, until, during

## LEVEL 1: Adding Objects (What?)

### Formula:
**Pronouns** + **V1/V1-3rd** + **(Determiner)** + **Object**

### Sample Sentences Using Actual Cards:

**Positive Examples:**
- **I** + **eat** + **pizza** = "I eat pizza" ‚úÖ
- **She** + **cooks** + **a** + **book** = "She cooks a book" ‚úÖ (grammatically correct even if logically odd)
- **They** + **watch** + **TV** = "They watch TV" ‚úÖ (plural object, no determiner)
- **He** + **reads** + **a** + **book** = "He reads a book" ‚úÖ (singular object, needs determiner)

**Negative Examples:**
- **I** + **don't** + **eat** + **pizza** = "I don't eat pizza" ‚úÖ
- **She** + **doesn't** + **cook** + **a** + **book** = "She doesn't cook a book" ‚úÖ

### ‚úÖ What to Look For:

1. **Subject-Verb Agreement:**
   - ‚úÖ **He/She/It** uses V1-3rd form (cooks, reads, goes)
   - ‚úÖ **I/You/We/They** uses V1 base form (cook, read, go)
   - ‚ùå WRONG: "He cook" or "They cooks"

2. **Determiner Usage:**
   - ‚úÖ Singular objects (book) ‚Üí needs determiner (a, an, the, my, your)
   - ‚úÖ Plural/uncountable objects (pizza, coffee, English, homework, TV, bus, music) ‚Üí NO determiner
   - ‚ùå WRONG: "She reads book" (missing determiner for singular)
   - ‚ùå WRONG: "I eat a pizza" (unnecessary determiner for plural/uncountable)

3. **Negative Sentences:**
   - ‚úÖ After "do not/does not" ‚Üí ALWAYS use V1 base form
   - ‚úÖ "I do not eat" (not "eats")
   - ‚úÖ "She does not cook" (not "cooks")
   - ‚ùå WRONG: "I do not eats" or "She does not cooks"

4. **Card Order:**
   - ‚úÖ Pronouns ‚Üí Helper Verb (if negative) ‚Üí V1 ‚Üí Determiner (if needed) ‚Üí Object
   - ‚ùå Cards in random order

## LEVEL 2: Adding Time Words (When?)

### Formula:
**Pronouns** + **V1/V1-3rd** + **Object** + **Time**

### Sample Sentences Using Actual Cards:

**Positive Examples:**
- **I** + **eat** + **pizza** + **every day** = "I eat pizza every day" ‚úÖ
- **She** + **works** + **now** = "She works now" ‚úÖ
- **They** + **watch** + **TV** + **usually** = "They watch TV usually" ‚úÖ
- **He** + **reads** + **a** + **book** + **tomorrow** = "He reads a book tomorrow" ‚úÖ

**Negative Examples:**
- **I** + **don't** + **eat** + **pizza** + **every day** = "I don't eat pizza every day" ‚úÖ
- **She** + **doesn't** + **work** + **now** = "She doesn't work now" ‚úÖ

### ‚úÖ What to Look For:

1. **Time Word Placement:**
   - ‚úÖ Time can go at end: "I eat pizza every day"
   - ‚úÖ Time can go at beginning: "Every day I eat pizza" (but cards usually show end placement)
   - ‚ö†Ô∏è Some awkward but acceptable: "He reads a book tomorrow" (future meaning with present tense)

2. **Subject-Verb Agreement:**
   - ‚úÖ Still check: He/She/It uses V1-3rd form
   - ‚úÖ Still check: Negative uses V1 base form after helper verb

3. **All Previous Checks:**
   - ‚úÖ Determiner usage (singular objects need it)
   - ‚úÖ Negative helper verb + V1 base form

## LEVEL 3: Adding Places (Where?)

### Formula:
**Pronouns** + **V1/V1-3rd** + **Object** + **Preposition** + **Place**

### Sample Sentences Using Actual Cards:

**Positive Examples:**
- **I** + **go** + **to** + **school** = "I go to school" ‚úÖ
- **She** + **works** + **at** + **home** = "She works at home" ‚úÖ
- **They** + **play** + **in** + **the** + **park** = "They play in the park" ‚úÖ (needs determiner with place)
- **He** + **studies** + **at** + **work** = "He studies at work" ‚úÖ

**Negative Examples:**
- **I** + **don't** + **go** + **to** + **school** = "I don't go to school" ‚úÖ
- **She** + **doesn't** + **work** + **at** + **home** = "She doesn't work at home" ‚úÖ

### ‚úÖ What to Look For:

1. **Preposition + Place:**
   - ‚úÖ Common combinations: "at home", "at school", "at work", "in the park", "to school"
   - ‚ö†Ô∏è Some places need determiner: "in the park", "at the school" (though "at school" also works)
   - ‚ùå WRONG: Missing preposition: "I go school"

2. **Preposition Choice:**
   - ‚úÖ "at" for locations (at home, at work, at school)
   - ‚úÖ "in" for contained spaces (in the park)
   - ‚úÖ "to" for direction (to school, to work, to home)

3. **All Previous Checks:**
   - ‚úÖ Subject-verb agreement
   - ‚úÖ Determiner usage
   - ‚úÖ Negative helper verb usage

## LEVEL 4: Past Simple (V2)

### Formula:
**Pronouns** + **V2** + **Object** + **Time**

### Sample Sentences Using Actual Cards:

**Positive Examples:**
- **I** + **ate** + **pizza** + **yesterday** = "I ate pizza yesterday" ‚úÖ
- **She** + **worked** + **last night** = "She worked last night" ‚úÖ
- **They** + **watched** + **TV** + **last night** = "They watched TV last night" ‚úÖ
- **He** + **read** + **a** + **book** + **yesterday** = "He read a book yesterday" ‚úÖ

**Negative Examples:**
- **I** + **didn't** + **eat** + **pizza** + **yesterday** = "I didn't eat pizza yesterday" ‚úÖ
- **She** + **didn't** + **work** + **last night** = "She didn't work last night" ‚úÖ

### ‚úÖ What to Look For:

1. **V2 Cards for Positive:**
   - ‚úÖ Use V2 cards (ate, worked, watched, went, studied, played, cooked, read, slept, cleaned, drove, talked, had, did, lived, needed, wanted)
   - ‚ùå WRONG: "I eat pizza yesterday" (should be "ate")

2. **CRITICAL: Negative Uses V1:**
   - ‚úÖ After "didn't" ‚Üí ALWAYS use V1 base form (eat, work, watch, NOT ate, worked, watched)
   - ‚úÖ "I didn't eat" (not "ate")
   - ‚úÖ "She didn't work" (not "worked")
   - ‚ùå WRONG: "I didn't ate" or "She didn't worked"

3. **Time Words:**
   - ‚úÖ Past simple often uses: **yesterday**, **last night**, **last week**
   - ‚úÖ Helps show it's past tense

4. **Subject-Verb Agreement (No 3rd person -s in past):**
   - ‚úÖ "He worked" (not "works" - that's present!)
   - ‚úÖ "She ate" (not "eats")

## LEVEL 5: Future Simple

### Formula:
**Pronouns** + **will** + **V1** + **Object** + **Time**

### Sample Sentences Using Actual Cards:

**Positive Examples:**
- **I** + **will** + **eat** + **pizza** + **tomorrow** = "I will eat pizza tomorrow" ‚úÖ
- **She** + **will** + **work** + **next weekend** = "She will work next weekend" ‚úÖ
- **They** + **will** + **watch** + **TV** + **tomorrow** = "They will watch TV tomorrow" ‚úÖ

**Negative Examples:**
- **I** + **won't** + **eat** + **pizza** + **tomorrow** = "I won't eat pizza tomorrow" ‚úÖ
- **She** + **won't** + **work** + **next weekend** = "She won't work next weekend" ‚úÖ

### ‚úÖ What to Look For:

1. **"will" Helper Verb:**
   - ‚úÖ Always use **will** card (not "going to" - not available)
   - ‚úÖ All subjects use same form: "I will", "She will", "They will"
   - ‚úÖ No subject-verb agreement needed!

2. **V1 Base Form After "will":**
   - ‚úÖ Always use V1 base form (eat, work, watch, NOT eats, works, ate, working)
   - ‚úÖ "I will eat" (not "eats" or "eating")
   - ‚ùå WRONG: "I will eats" or "I will ate"

3. **Negative: "won't":**
   - ‚úÖ Use **won't** card
   - ‚úÖ Still V1 base form after: "I won't eat"
   - ‚ùå WRONG: "I won't eats"

4. **Future Time Words:**
   - ‚úÖ Often uses: **tomorrow**, **next weekend**, **next week**

## LEVEL 6: Present Continuous - Introduction

### Formula:
**Pronouns** + **To Be** + **V1-ing**

### Sample Sentences Using Actual Cards:

**Positive Examples:**
- **I** + **am** + **eating** = "I am eating" ‚úÖ
- **She** + **is** + **cooking** = "She is cooking" ‚úÖ
- **They** + **are** + **watching** = "They are watching" ‚úÖ
- **He** + **is** + **studying** = "He is studying" ‚úÖ

**Negative Examples:**
- **I** + **am not** + **eating** = "I am not eating" ‚úÖ
- **She** + **isn't** + **cooking** = "She isn't cooking" ‚úÖ
- **They** + **aren't** + **watching** = "They aren't watching" ‚úÖ

### ‚úÖ What to Look For:

1. **To Be Matching:**
   - ‚úÖ **I** ‚Üí **am** or **am not**
   - ‚úÖ **You/We/They** ‚Üí **are** or **aren't**
   - ‚úÖ **He/She/It** ‚Üí **is** or **isn't**
   - ‚ùå WRONG: "I is eating" or "They am watching"

2. **V1-ing Form:**
   - ‚úÖ Must use V1-ing cards (eating, cooking, watching, not eat, cook, watch)
   - ‚ùå WRONG: "I am eat" (need "eating")

3. **No Helper Verbs in Positive:**
   - ‚úÖ Positive continuous: Pronouns + To Be + V1-ing
   - ‚ùå WRONG: "I do am eating" (don't use helper verbs with To Be)

## LEVEL 7: Present Continuous - Full Sentences

### Formula:
**Pronouns** + **To Be** + **V1-ing** + **Object** + **Preposition** + **Place** + **Time**

### Sample Sentences Using Actual Cards:

**Positive Examples:**
- **I** + **am** + **preparing** + **a** + **report** + **at** + **work** + **now** = "I am preparing a report at work now" ‚úÖ
- **She** + **is** + **cooking** + **pizza** + **at** + **home** + **every day** = "She is cooking pizza at home every day" ‚úÖ
- **They** + **are** + **watching** + **TV** + **in** + **the** + **park** + **tomorrow** = "They are watching TV in the park tomorrow" ‚úÖ (future planned event)

**Negative Examples:**
- **I** + **am not** + **preparing** + **a** + **report** + **at** + **work** + **now** = "I'm not preparing a report at work now" ‚úÖ
- **She** + **isn't** + **cooking** + **pizza** + **at** + **home** + **every day** = "She isn't cooking pizza at home every day" ‚úÖ

### ‚úÖ What to Look For:

1. **All Previous Checks:**
   - ‚úÖ To Be matching (I am, She is, They are)
   - ‚úÖ V1-ing form used
   - ‚úÖ Determiner usage for singular objects
   - ‚úÖ Preposition + Place correctly combined

2. **Complete Sentences:**
   - ‚úÖ Has Object (even if just one word like "pizza")
   - ‚úÖ Optional but encouraged: Place and/or Time
   - ‚ö†Ô∏è Minimum: Pronouns + To Be + V1-ing + Object

3. **Future Meaning with Present Continuous:**
   - ‚úÖ "They are watching TV tomorrow" = planned/scheduled future event
   - ‚úÖ This is correct! Present continuous can show future plans

## LEVEL 8: Past Continuous

### Formula:
**Pronouns** + **To Be (was/were)** + **V1-ing** + **Connector** + **Second Event**
OR
**Pronouns** + **To Be (was/were)** + **V1-ing** + **Time**

### Sample Sentences Using Actual Cards:

**With Connectors:**
- **I** + **was** + **cooking** + **when** + **he** + **came** + **home** = "I was cooking when he came home" ‚úÖ
- **She** + **was** + **working** + **while** + **I** + **was** + **studying** = "She was working while I was studying" ‚úÖ
- **They** + **were** + **watching** + **TV** + **before** + **she** + **left** = "They were watching TV before she left" ‚úÖ

**With Time:**
- **I** + **was** + **eating** + **pizza** + **yesterday** = "I was eating pizza yesterday" ‚úÖ
- **They** + **were** + **meeting** + **clients** + **last night** = "They were meeting clients last night" ‚úÖ

**Negative:**
- **I** + **wasn't** + **cooking** + **when** + **he** + **came** + **home** = "I wasn't cooking when he came home" ‚úÖ

### ‚úÖ What to Look For:

1. **To Be Matching (Past):**
   - ‚úÖ **I/He/She/It** ‚Üí **was** or **wasn't**
   - ‚úÖ **You/We/They** ‚Üí **were** or **weren't**
   - ‚ùå WRONG: "I were cooking" or "They was watching"

2. **CRITICAL: Connector or Time Required:**
   - ‚úÖ Affirmative past continuous MUST have:
     - Either a **Connector** (when, while, before) + second event
     - OR a **Time** word (yesterday, last night)
   - ‚ùå WRONG: "I was cooking" (incomplete - needs connector or time)
   - ‚úÖ CORRECT: "I was cooking when he came" OR "I was cooking yesterday"

3. **Connector Cards:**
   - ‚úÖ Available connectors: **when**, **while**, **before**, **as**, **after**, **until**, **during**
   - ‚úÖ After connector, need a second subject + verb

4. **V1-ing Form:**
   - ‚úÖ Still use V1-ing cards (cooking, working, watching)

## LEVEL 9: Future Continuous

### Formula:
**Pronouns** + **will** + **be** + **V1-ing** + **Object** + **Time**

### Sample Sentences Using Actual Cards:

**Positive Examples:**
- **I** + **will** + **be** + **preparing** + **a** + **report** + **tomorrow** = "I will be preparing a report tomorrow" ‚úÖ
- **She** + **will** + **be** + **meeting** + **clients** + **next weekend** = "She will be meeting clients next weekend" ‚úÖ
- **They** + **will** + **be** + **watching** + **TV** + **at 3 PM** + **tomorrow** = "They will be watching TV at 3 PM tomorrow" ‚úÖ

**Negative Examples:**
- **I** + **won't** + **be** + **preparing** + **a** + **report** + **tomorrow** = "I won't be preparing a report tomorrow" ‚úÖ

### ‚úÖ What to Look For:

1. **"will" + "be" Required:**
   - ‚úÖ Must have both: **will** + **be** (to be card with "be" not available, but concept is clear)
   - ‚ö†Ô∏è Note: "be" is not a separate card - this is a concept to explain
   - ‚úÖ Pattern: will + be + V1-ing

2. **V1-ing After "be":**
   - ‚úÖ Must use V1-ing cards (preparing, meeting, watching)
   - ‚ùå WRONG: "I will be prepare" (need "preparing")

3. **Future Time:**
   - ‚úÖ Uses future time words: **tomorrow**, **next weekend**

4. **All Subjects Same:**
   - ‚úÖ No subject-verb agreement: "I will be", "She will be", "They will be" all work the same

## LEVEL 10: Negative Statements (Review)

### What to Check:

**Simple Present Negative:**
- ‚úÖ **don't/doesn't** + V1 base form
- ‚úÖ Check: He/She/It uses "doesn't", others use "don't"
- ‚ùå WRONG: "I don't eats" or "She doesn't cooks"

**Present Continuous Negative:**
- ‚úÖ **am not/isn't/aren't** + V1-ing
- ‚úÖ Check: To Be matches subject
- ‚ùå WRONG: "I isn't eating"

**Past Continuous Negative:**
- ‚úÖ **wasn't/weren't** + V1-ing
- ‚úÖ Check: I/He/She/It uses "wasn't", others use "weren't"
- ‚úÖ Still needs connector or time!

**Past Simple Negative:**
- ‚úÖ **didn't** + V1 base form (NOT V2!)
- ‚ùå WRONG: "I didn't ate" (should be "eat")

**Future Simple Negative:**
- ‚úÖ **won't** + V1 base form
- ‚ùå WRONG: "I won't eats"

**Future Continuous Negative:**
- ‚úÖ **won't** + **be** + V1-ing
- ‚ùå WRONG: "I won't be prepare"

## Quick Check Guide

### Visual Checks:
- ‚úÖ **Color pattern matches formula** in scaffold guide
- ‚úÖ **Cards read left-to-right** as a sentence
- ‚úÖ **Not too many cards** (3-8 cards usually)

### Common Mistake Checklist:
- [ ] Subject-verb agreement (He/She/It needs V1-3rd in present)
- [ ] Determiner with singular objects
- [ ] Negative uses V1 base form (not V1-3rd or V2)
- [ ] Past continuous has connector or time
- [ ] To Be matches subject (I am, She is, They are)
- [ ] V1-ing form used in continuous tenses
- [ ] V2 used in past simple positive (not V1)
- [ ] "will" + V1 (base form) in future tenses

## Teaching Tips

1. **Start Simple:** Level 1 only for first lesson
2. **Build Gradually:** Add one level per session
3. **Use Color Memory:** "Green cards = verbs, but different shades mean different forms"
4. **Ask Questions:** "Who is doing it? What are they doing? When? Where?"
5. **Celebrate Mistakes:** Wrong sentences are learning opportunities!
